# IWarnChrome
当访问指定网页,通知一个消息给自己.

# 安装方法

请使用最新版Chrome浏览器.       

	1.打开Chrome,找到右侧菜单->更多工具->扩展程序;     
	2.将iwarn.crx文件拖动到扩展程序页面,即可安装;     
	3.点击图标即可添加网址(右键->选项->可以更新或者修改网址).    

**请注意,因未提交到Chrome应用商店,扩展在重启浏览器后会被禁用(可用作学习)!**

# 特征
	1.HTML5的LocalStorage(未使用Chrome Storage是遇到一部分问题);   
	2.ES6新标准实现的JavaScript;    
	3.代码结构化,可自由修改,自由新增.
